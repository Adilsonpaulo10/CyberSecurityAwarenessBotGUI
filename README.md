# 🔒 Cybersecurity Awareness Chatbot GUI (WPF Application)

## 📌 Project Overview
This project is a Windows Presentation Foundation (WPF) chatbot application designed to educate users about cybersecurity concepts in an interactive and user-friendly way.

The chatbot simulates basic Natural Language Processing (NLP) techniques and provides responses related to cybersecurity topics such as phishing, passwords, VPNs, malware, and privacy.

It also includes memory, sentiment detection, and a graphical user interface (GUI).

---

## 🧠 Features

### 💬 Chatbot Features
- Keyword-based response system
- Basic NLP simulation (intent recognition)
- Sentiment detection (e.g., worried, curious, frustrated)
- Memory system (remembers user interests)
- Error handling for invalid inputs

### 🖥️ GUI Features (WPF)
- Modern chat interface using WPF
- RichTextBox chat display
- Text input box for user messages
- Send button for interaction
- Styled layout with readable UI

### 🔊 Multimedia Features
- WAV voice greeting on startup
- ASCII art logo displayed in chat window

---

## 🧱 Technologies Used
- C#
- .NET 8 WPF
- Visual Studio
- System.Media (SoundPlayer)
- Dictionary collections for memory and responses

---

## 📂 Project Structure

- `MainWindow.xaml` → GUI layout
- `MainWindow.xaml.cs` → UI logic and event handling
- `ChatbotEngine.cs` → Chatbot intelligence (responses, memory, NLP)
